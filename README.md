## 👋 项目介绍

MonkeyCode 是一个开源的 AI 编程助手，旨在通过智能代码补全、错误检测、代码重构和文档生成等功能，提升开发者的编码效率和代码质量。无论您是经验丰富的专业人士还是编程新手，MonkeyCode 都能成为您开发流程中不可或缺的强大工具。
